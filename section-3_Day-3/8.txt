Understanding the FormData Object
FormData: This is a built-in JavaScript object that allows you to easily construct a set of key/value pairs
 representing form fields and their values.
Usage: It is particularly useful for handling form submissions, especially when you need to collect data 
from a form that includes file uploads.

       const formData = new FormData(event.currentTarget)
This instance then allows you to easily access and manipulate the data submitted through the form.
