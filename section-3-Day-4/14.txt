https://www.npmjs.com/package/react-markdown


react-markdown is a React component that safely renders Markdown text as React elements.
We use it to display user-written or dynamic Markdown content in React apps.
It avoids security risks like XSS by not using dangerouslySetInnerHTML.
It also lets you customize rendering with plugins and your own React components.


dangerouslySetInnerHTML is a React property used to insert raw HTML directly into the DOM.
Itâ€™s used when you need to render HTML strings that React would normally escape.
The risk is that it can expose your app to XSS (security) attacks if the HTML is untrusted.
Because of this danger, React requires an explicit name to warn developers before using it.
